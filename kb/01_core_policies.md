# Core Security Policies (Startup-Ready)

Minimum policy pack:
- Access Control (MFA, least privilege, joiner/mover/leaver)
- Data Classification & Handling (PII/PHI/PCI rules)
- Incident Response (roles, escalation, runbooks)
- Vulnerability & Patch Management (scan cadence, patch SLAs)
- Backup & Recovery (3-2-1, restore tests)
- Vendor Risk (SOC2 reports, DPAs, breach clauses)
