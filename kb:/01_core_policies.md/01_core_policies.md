Starter security policy set for small businesses:

1) Information Security Policy: ownership, governance, objectives, risk approach.
2) Access Control Policy: least privilege, MFA, RBAC, access reviews, joiner/mover/leaver.
3) Data Classification & Handling: define data types (public/internal/confidential), handling rules, encryption expectations.
4) Secure Development/Change Management: reviews, approvals, separation of duties, rollback, SDLC checkpoints.
5) Logging & Monitoring Standard: what to log, alerting, retention, and review cadence.
6) Vendor Risk Management: evaluate SaaS/cloud vendors, contracts, security questionnaires, DPAs.
7) Incident Response Plan: severity levels, escalation, containment, forensics, notifications, lessons learned.
8) Backup & Recovery: RPO/RTO targets, restore testing, immutable backups for critical systems.
9) Acceptable Use Policy: employee expectations and prohibited behaviors.
