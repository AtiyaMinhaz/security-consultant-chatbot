This chatbot provides security program guidance for early-stage businesses.

Core outputs:
- Recommended security policies and standards
- Practical controls to implement first
- Evidence to retain for customers/investors/audits

Common frameworks referenced:
- NIST CSF (Identify, Protect, Detect, Respond, Recover)
- CIS Controls (implementation checklist)
- ISO/IEC 27001 (governance and ISMS concepts)
- SOC 2 (assurance-style controls and evidence expectations)
